The calculator is a simple program to calculate weapons/forcerank.

Run it, find the weapons.mcm or force.mcm file and open it (Included in helpfiles folder)

Click the items you don't want to keep, and press space.
Then go to your JK2 settings file and add:
g_forcepowerdisable <paste number here>
OR
g_weapondisable <paste number here>


To calculate admin powers, open admin_powers<PowerNum>.mcm and paste the result in the account file.
